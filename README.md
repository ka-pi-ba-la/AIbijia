## 网站定位

目前AI账号的价格很混乱，同一种类型的账号，不同平台、不同代理商的价格很“五彩缤纷“。

比如ChatGPT plus CDK，有的卖30，有的卖40，还有的卖60。

但如果仔细对比它们的提货渠道，就会发现，它们是从同一个源头拿的货，只不过各自的加价不同，账号本质上没什么区别。
